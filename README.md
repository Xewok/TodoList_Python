# TodoList - Консольное приложение

Простое консольное приложение для управления задачами на Python.

## Описание

Приложение позволяет:
- Создавать и управлять списком задач
- Отмечать выполненные задачи
- Удалять ненужные задачи
- Сортировать задачи по статусу
- Автоматически сохранять данные

## Быстрый старт
